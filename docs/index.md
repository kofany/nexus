## Overview

NexusIRC is a modern web IRC client featuring:

- Persistent IRC connections
- irssi FE-Web protocol support
- WeeChat relay bridge
- Multi-user, always-on architecture
- Encrypted message storage

For a high-level overview, refer to the [README](../README.md).
